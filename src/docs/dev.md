# 开发文档

面向第三方开发者的接口与参数说明。以下功能均可在不修改源码的情况下直接使用。

---

## URL 参数

### 全局参数

以下参数可附加在任意页面的 URL 查询字符串中，效果作用于全站。

#### 1. `theme` — 强制主题模式

| 参数值         | 效果                     |
| -------------- | ------------------------ |
| `?theme=dark`  | 强制切换为深色模式       |
| `?theme=light` | 强制切换为浅色模式       |
| `?theme=auto`  | 跟随系统主题（默认行为） |

**示例：**

- `https://tool.lonzov.top/settings?theme=dark`：以深色模式打开设置页
- `https://tool.lonzov.top/?theme=light`：以浅色模式打开首页

> 通过 URL 设置的主题会覆盖浏览器的 localStorage 记录，刷新或下次访问时继续生效。

---

### 首页参数

以下参数仅作用于首页（`/`）。

#### 1. `search` — 站内搜索

| 参数值             | 效果                                   |
| ------------------ | -------------------------------------- |
| `?search=<关键词>` | 进入首页后自动填充搜索框并执行站内搜索 |

**示例：**

- `https://tool.lonzov.top/?search=id`：打开首页并搜索 "id"
- `https://tool.lonzov.top/?search=命令助手`：打开首页并搜索 "命令助手"

**行为说明：**

1. 检测到 `search` 参数时，搜索引擎强制切换为"站内搜索"
2. 搜索框自动填入参数值，触发卡片过滤
3. 地址栏会随搜索内容实时更新（静默替换，不触发页面重载）

> **建议：** 当关键词包含中文、空格或特殊符号时，推荐先经过 URL 编码，例如 `命令助手` → `%E5%91%BD%E4%BB%A4%E5%8A%A9%E6%89%8B`、`hello world` → `hello%20world`
