## 导言

欢迎使用"小舟工具箱"网站，域名 tool.lonzov.top（以下简称"本网站"）。我们深知个人信息保护的重要性，并致力于保护您的隐私安全。本隐私政策旨在向您清晰地说明我们如何处理您的个人信息，包括Cookie和本地存储等技术的使用情况。

请知悉，本网站为纯静态网站，**本身不会主动收集、存储或使用您的任何个人身份信息，也不会设置任何HTTP Cookie**。为提升用户体验，本网站会使用浏览器本地存储技术（如LocalStorage、SessionStorage）保存部分非敏感的用户偏好设置；同时，作为PWA（渐进式Web应用）站点，会使用Service Worker缓存资源以支持离线访问。此外，为进行数据分析，我们接入了第三方统计工具"51.la"，该工具可能会使用Cookie及相关技术收集部分匿名化浏览数据。请您仔细阅读本隐私政策，了解我们对您信息的处理方式。

## Cookie与本地存储说明

### 本网站使用的本地存储（非Cookie）
本网站**不设置任何HTTP Cookie**，但会合理使用浏览器客户端存储技术，所有数据均仅保存在您的设备本地，**不会上传至服务器**，包括：

- **用户界面偏好**：如深色/浅色模式选择
- **交互状态记录**：如部分提示弹窗的“不再显示”状态
- **本地版本标识**：用于检测缓存更新
- **PWA运行所需缓存**：
  - Service Worker注册信息
  - 静态资源缓存
  - 应用清单（manifest）相关数据

> 所有上述存储内容均为功能性、非追踪性质，且**不包含任何个人身份信息**。您可随时通过浏览器“清除站点数据”功能完全删除这些本地数据。

### 第三方统计工具使用的Cookie
- **51.la统计工具**：我们接入的 51.la 统计服务可能会在您的设备上放置 Cookie，用于：
  - 区分唯一访客
  - 记录会话时长
  - 识别流量来源
  - 优化统计准确性
- 所有通过 51.la 收集的数据均**经过匿名化处理**，不会与您的个人身份直接关联
- 51.la 对 Cookie 的使用受其[隐私政策](https://www.51.la/about/privacy/)约束

## 个人信息收集范围与方式

### 通过51.la统计工具收集的信息
51.la 可能会自动收集并处理以下信息：
- IP 地址
- 用户代理 (User-Agent) 信息
- 访问的 URL 和页面路径
- 访问时间和停留时长
- 通过 Cookie 生成的匿名访客标识符

> 请注意：**本地存储中的数据（如主题设置）不会被发送给51.la或任何第三方**。

### GitHub Pages 收集的个人信息  
本站托管在 GitHub Pages，其出于安全目的，无论访问者是否已登录 GitHub ，都会记录并存储访问者的 IP 地址。  
 有关 GitHub 的安全做法的详细信息，请参阅 [GitHub 隐私声明](https://docs.github.com/zh/site-policy/privacy-policies/github-general-privacy-statement)。

## 信息使用目的

- **本地存储数据**：仅用于在您设备上提供个性化体验（如记住主题偏好）、优化交互流畅性（如避免重复弹窗），纯客户端使用。
- **51.la收集的匿名化数据**仅用于：
  - 分析网站流量和用户行为
  - 优化网站内容和用户体验
  - 了解网站功能使用情况
  - 改进网站性能和结构

## 您的选择与控制权

### 管理本地存储与缓存
- 您可随时在浏览器中清除“站点数据”或“缓存与Cookie”，将重置所有本地设置（需注意，部分页面如“T显可视化编辑器”会利用本地缓存储存您的编辑历史，清除前请**务必对这些数据进行备份**避免造成不必要的损失）
- 禁用 JavaScript 将阻止本地存储功能（但会导致大部分功能无法使用）
- 无痕/隐私浏览模式下，关闭窗口后本地数据将自动清除

### 管理第三方Cookie与追踪
您可通过以下方式阻止 51.la 的数据收集：
1. **浏览器设置**：
   - 阻止第三方Cookie
   - 启用"无痕/隐私浏览"模式
   - 配置Cookie例外，拒绝`*.51.la`
2. **拦截工具**：
   - 安装Ublock Origin、Privacy Badger等插件
   - 将51.la相关域名加入拦截列表
3. **退出追踪**：
   - 启用"Do Not Track"（DNT），又名 禁止跟踪、跟踪防护、跟踪保护

> **重要提示**：阻止 51.la 统计或清除本地存储**不会影响**网站核心功能的正常使用。主题偏好、弹窗记忆、编辑历史等功能仅作为便利性增强，非必需。
